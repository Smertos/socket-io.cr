# SocketIO

Small wrapper over [HTTP::WebSocket](http://ru.crystal-lang.org/api/HTTP/WebSocket.html)

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  socket_io:
    github: forsaken1/socket-io.cr
```


## Usage


```crystal
require "socket_io"
```
